Columnmerge
===========

Assignment
----------

Generate a list that can combines 1 value out of each column.
Not every column has to be used and a column can only be used once.

* Column 1: A1, A2
* Column 2: B1, B2
* Column 3: C1, C2

Sample output: (The order of the output is irrelevant.)

> A1  
> B1  
> C1  
> A1 B1  
> A1 C1  
> A1 B1 C1  
> A1 C1 B1  
> ...
